# Project Title: Disney Movies Data Explorer

Authors: Cadet Ketron, Cadet Preti, Cadet Cresci
Date: Novemeber 20, 2025

## Overview

This game is loosely based off of the old flash game  "Stick War" with the spin being cats
with guns. This game was developed for the CY300 "In Class Project" over the course of ~ 2 months.


## Files

- `main.py`: The main script to run the game.
- `button.py`: A module for handeling and creating text-based buttons buttons.
- `imagebutton.py`: A module for handeling and creating image-based buttons.
- `cat_characters.py`: The class structure for the in-game characters.
- `README.md`: Instructions for running and using the program (this file).
- `settings.py`: The script to handle the in-game settings.
- `shop.py`: The script to handle the in-game shop.
- `towers.py`: The class structure for the in-game towers.
- `overlay_text.py`: The script to handle text overlaying the game.
- `money.py`: The script to handle the in-game money.
- `levels.py`: The script for handeling in-game levels.
- `gems.py`: The script for handeling the in-game gem currency
- `game_stats.py`: The script for handeling the in-agme stats.
- `bugs.txt`: The text file for reporting non-game breaking bugs to be fixed on a later day.


## Running the Program

To run the program:

1. Open a terminal or command prompt.
2. Navigate to the folder containing the project files.
3. Type pip install pygame into the command prompt.
4. Run the program with the following command: python main.py

## Dependencies

The program requires Python 3.11.9 or higher. Ensure the following libraries are
installed:

- `pygame`
- `sys` (built-in)
- `random` (built in)
